## Setup

* `git clone` this repo
* `cd` into it.
* `npm install`
* `cd client && npm install`

*You need all of the keys for the API. Contact me so I can send it to you.*

## To make it run

* `npm start` from project root or top level folder
* `npm start` inside of `/client`
